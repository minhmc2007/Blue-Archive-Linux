About Blue Archive Linux

This isn't just another Linux distro with anime wallpapers. This is a complete Blue Archive computing experience. It's perfect for Blue Archive fans, power users who love minimal systems, and developers seeking a clean environment.

Quick Start

Method 1: ISO Installation (Recommended)

Download the latest ISO from the releases page.

Flash to a USB drive. For Linux:

sudo dd if=Blue_Archive_Linux*.iso of=/dev/sdX bs=4M status=progress

For Windows, use a tool like Rufus. Then, boot from the USB and follow the installer.

Method 2: Manual Build (Advanced)

Clone the repository and run the build script:

git clone https://github.com/minhmc2007/Blue-Archive-Linux.git
cd Blue-Archive-Linux/blue_archive_linux/stable
sudo bash build.sh

bluepm Package Manager

Meet bluepm - our custom package manager designed specifically for Blue Archive Linux. It's fast, secure, and focused on providing curated packages for the perfect BA experience.

# Install BA-specific packages
bluepm install bal-xfce bal-kde-plasma

# Search for themes
bluepm search bal-gnome

# Keep your system updated
bluepm update-cache

System Specifications

Base System

  • Distribution: Debian "bookworm"
  • Architecture: amd64
  • Bootloader: Syslinux (BIOS), GRUB (EFI)
  • Image Type: Hybrid ISO

Included Software

  • Core: Linux kernel, systemd
  • Shell: fish
  • Editors: nano, vim-tiny
  • System Tools: sudo, htop
  • Networking: NetworkManager, wireless-tools, iw, curl, wget
  • Filesystem: fdisk, gdisk, parted, dosfstools, btrfs-progs, e2fsprogs
  • Other: git, python3-requests, python3-pip

Build Process

Blue Archive Linux is built using live-build, a set of scripts that allows for the creation of custom Debian-based live systems. The build process is defined by the configuration files in the blue_archive_linux/config directory.

The build.sh script automates the build process. It performs the following steps:

  1. sudo lb clean: Cleans the build environment.
  2. sudo lb build: Builds the live image.
  3. mv live-image-amd64.hybrid.iso ...: Renames the output ISO file with a timestamp.
